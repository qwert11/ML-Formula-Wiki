# ML Formula Wiki

Онлайн-шпаргалка по математическим формулам машинного обучения из книги
**Себастьяна Рашки «Python и машинное обучение»** (стр. 61–100).
Формулы идут в порядке появления в книге с указанием страницы — сайт работает
как конспект-компаньон при чтении.

**Сайт:** https://qwert11.github.io/ML-Formula-Wiki/

## Что внутри

19 формул в 4 блоках, каждая — карточка единого формата:

| Поле | Содержание |
|------|-----------|
| Формула | Запись в LaTeX (рендер KaTeX) |
| Краткое название | Общепринятое имя |
| Кто и когда | Автор и год появления |
| Параметры | Расшифровка каждого символа |
| Что делает | Детальное описание роли формулы в ML |
| Реальные примеры | 2–3 применения на практике |
| Источник | Глава и страница книги |

Блоки: **A** — основы обучения (стандартизация, SGD), **B** — логистическая
регрессия, **C** — метод опорных векторов, **D** — деревья решений.

## Устройство

Ванильная статика без сборки — деплой на GitHub Pages простым пушем:

```text
index.html          — страница-шпаргалка
styles.css          — стили (mobile-first)
app.js              — рендер карточек, оглавление, поиск, копирование LaTeX
data/formulas.js    — все формулы (единственный источник данных)
plan/               — проектная документация
```

Единственная внешняя зависимость — [KaTeX](https://katex.org/) с CDN.

## Как добавить формулу

1. Добавить объект в массив `FORMULAS` в `data/formulas.js`
   (схема описана в `plan/tz/02_data_model.md`).
2. Проверить, что LaTeX рендерится KaTeX без ошибок.
3. Закоммитить и запушить — сайт обновится сам.

## Источник и права

Все формулы — из книги С. Рашки «Python и машинное обучение» с указанием
страниц (книжная нумерация). Публикуются только сами формулы (математические
факты) и собственные описания; текст книги не воспроизводится.
