# 03. Content source — источник и извлечение формул

## Источник

- Книга: **Себастьян Рашка, «Python и машинное обучение»** (рус. издание).
- Файл: `plan/books/Рашка Себастьян - Python и машинное обучение.pdf`
  (420 PDF-страниц; локально, в git не попадает).
- Диапазон: **вся книга** (главы 2–13, книжные стр. 44–371).
  Исходный диапазон первого этапа — стр. 61–100 (блоки A–D); 2026-07-03
  заказчик расширил охват на всю книгу («все формулы те что вначале и до
  конца»), см. `plan/faq.md`.
- Соответствие нумераций: **PDF-страница = книжная страница + 1**
  (книжная стр. 61 = PDF стр. 62).

## Правила извлечения

1. Идём по страницам строго по порядку, выписываем каждую пронумерованную или
   выделенную математическую формулу (нумерации формул в книге нет — критерий:
   формула вынесена отдельной строкой или является определением).
2. Не берём: промежуточные шаги выводов (например, цепочку вывода частной
   производной SSE), примеры арифметики, фрагменты кода.
   Итог вывода (само правило/определение) — берём.
3. Формула, начатая до стр. 61 или продолжающаяся после стр. 100, берётся,
   только если её определение попадает в диапазон. Пограничные случаи — в
   `plan/faq.md`.
4. Текст из PDF извлекается с OCR-мусором — LaTeX набирается вручную по
   визуальной сверке со страницей книги, не копипастом.
5. Поля `discovered` (год, автор) и `examples` книга покрывает частично —
   добираем из внешних источников, проверяя даты по двум независимым.
